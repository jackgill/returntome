NAME

R2M::Conf -- read encrypted configuration files.

SYNOPSIS

my $conf = read_conf($conf_file, $key);

DESCRIPTION

Uses Config::Tiny to read ini-style config files.

SUBROUTINES

read_conf

Arguments:

  • Name of configuration file
  • Decryption key

Returns:

  • Config::Tiny object (hashref)

DEPENDENCIES

  • R2M::Crypt
  • Config::Tiny
